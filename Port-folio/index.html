Cascading Style Sheet(CSS), is a language used to style the presentation of HTML documents. It allows web developers to control the layout, colors, fonts, and other visual aspects of a webpage. React is a Javascript open source framework or library particularly used for building interfaces, single-page applications. Developed and maintained by facebook, React allows developers to create reuseable UI components that update efficiently when the underlying data changes. React uses a declerative approach, where developer describe how the UI should look base on the current state, and React take care of the updating the DOM to match this efficiently. Its component-based architecture promotes modularity, Making it easier to manage and scale complex applications. React is often used in conjunction with other liberaries and frameworks, such as Redux for state management and React Router for handling navigation.

PHP is a server-side scripting language commonly used for web development. It is particularly well-suited for building dynamic web pages and web applications. PHP code is executed on the server, generating HTML content that is then sent to clients web browser, which means users only sees the HTML output and not the PHP code itself. PHP can interact with databases, handle form data, manage sessions, and perform various other server-side tasks, making it a versatile tool for building interactive and data driven websites. it is supported by most web hosting servers and has a large and active community, contributing to its widespread use in web development.
